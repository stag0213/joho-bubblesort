# バブルソート学習サイト

高等学校の情報Ⅰを学ぶ高校生向けの、バブルソートアルゴリズムを視覚的に理解できる動的な学習Webサイトです。

## 📚 プロジェクト概要

このサイトは、バブルソートの仕組みを視覚的に理解し、インタラクティブに学習できる教育用Webアプリケーションです。アニメーション、ステップバイステップの実行、クイズ、チャレンジ問題など、多彩な機能を提供しています。

## ✨ 主な機能

### 1. 学習タブ
- **バブルソートの基本概念の説明**
  - わかりやすい日本語での解説
  - アルゴリズムの手順を3ステップで説明
  - 視覚的なステップカードで理解を促進

- **時間計算量の説明**
  - 最悪計算量、平均計算量、最良計算量の提示
  - O記法による表記

- **学習のポイント**
  - バブルソートの特徴と実用性についての情報

### 2. 視覚化タブ（メイン機能）
- **配列の入力・編集機能**
  - カンマ区切りでの自由な配列入力
  - クイック選択ボタン（小・標準・大きい配列）
  - ランダム配列生成機能
  - 入力検証（2-15要素、数値チェック）

- **アニメーション制御**
  - 自動ソート実行（開始/一時停止/リセット）
  - ステップバイステップ実行（1ステップずつ進める）
  - 速度調整スライダー（100ms～2000ms）
  - 状態管理（実行中、一時停止、完了）

- **視覚化表示**
  - バーチャートによる配列の視覚化
  - カラーコーディング：
    - 青：比較中の要素
    - オレンジ：交換中の要素
    - 緑：ソート済みの要素
    - グレー：未処理の要素
  - スムーズなアニメーション効果
  - 要素の値を表示

- **統計情報のリアルタイム表示**
  - パス回数
  - 比較回数
  - 交換回数
  - 現在の状態

- **凡例表示**
  - 各色の意味を視覚的に説明

### 3. コード例タブ
- **基本的なバブルソート（Python）**
  - 標準的な実装
  - 詳細なコメント付き
  - 使用例の提示

- **最適化されたバブルソート**
  - 早期終了の実装
  - 効率化の説明

- **詳細情報を表示するバブルソート**
  - 各ステップの詳細出力
  - 比較・交換回数のカウント
  - デバッグや学習に最適

- **練習問題**
  - 4つの実践的な練習課題
  - 段階的な難易度設定

### 4. クイズタブ
- **理解度チェッククイズ（全5問）**
  - 問題1: 最初のパスでの動作
  - 問題2: 時間計算量
  - 問題3: 具体的なソート過程
  - 問題4: 最良計算量の条件
  - 問題5: パス回数の計算

- **即座のフィードバック**
  - 正解・不正解の表示
  - 正解の強調表示
  - 解説メッセージ

- **結果サマリー**
  - 正解数と全体の結果
  - パフォーマンスに応じた評価メッセージ
  - もう一度挑戦機能

- **手動ソートチャレンジ**
  - ランダムに生成された配列を手動でソート
  - 隣接要素のみ交換可能（バブルソートのルール）
  - ユーザーの交換回数と最小交換回数の表示
  - ヒント機能
  - リセット機能
  - 完了時の評価

## 🚀 使い方

### 基本的な使い方

1. **学習タブ**で基礎知識を習得
2. **視覚化タブ**でアルゴリズムの動作を確認
3. **コード例タブ**で実装方法を学習
4. **クイズタブ**で理解度をチェック

### 視覚化機能の使い方

1. 配列入力欄に数値を入力（カンマ区切り）
2. 「配列をセット」ボタンをクリック
3. 速度スライダーでアニメーション速度を調整
4. 「開始」ボタンでソート開始
5. または「1ステップ実行」で1ステップずつ確認

### チャレンジの遊び方

1. 「チャレンジ開始」をクリック
2. 隣接する2つのバーをクリックして交換
3. すべて昇順に並べたらクリア
4. 最小交換回数でクリアを目指そう

## 🌟 特徴的な実装

### ジェネレータ関数による制御
バブルソートの各ステップをジェネレータ関数で実装し、柔軟なアニメーション制御を実現しています。比較を右から左へ進めることで、小さい値が左に「沈んで」いく様子を視覚的に表現しています。

### 状態管理
実行状態（実行中、一時停止、ステップモード）を適切に管理し、ボタンの有効/無効を自動制御しています。

### レスポンシブな視覚化
配列のサイズに応じてバーの幅と高さを動的に調整し、常に最適な表示を提供します。
