バブルソートとは？

バブルソートは、隣り合う要素を比較して交換を繰り返すことで、配列を並び替える（ソートする）アルゴリズムです。

配列の右端から左へと比較を進めていき、小さい値が左側に「沈んで」いく様子から、「バブルソート」と呼ばれています。

アルゴリズムの手順

1

配列の右端から左へ隣り合う要素を比較

左の要素が右の要素より大きければ交換する

2

配列の左端まで繰り返す

1回のパスで最小値が左端に移動する

3

全体を繰り返す

配列全体がソートされるまで繰り返す

時間計算量

最悪計算量: O(n²)
平均計算量: O(n²)
最良計算量: O(n)

ポイント: バブルソートはシンプルでわかりやすいアルゴリズムですが、データ量が多いと処理時間がかかります。学習用には最適ですが、実際のプログラムでは他の効率的なソートアルゴリズムが使われることが多いです。

バブルソートの視覚化

パス回数: 0
比較回数: 0
交換回数: 0
状態: 未開始
比較中
交換中
ソート済み
未処理

Pythonでのコード例

基本的なバブルソート

def bubble_sort(arr):
    """
    バブルソートのアルゴリズム
    
    引数:
        arr: ソートする配列
    """
    n = len(arr)
    
    # 外側のループ: パス回数
    for i in range(n):
        # 内側のループ: 隣接要素の比較
        for j in range(0, n - i - 1):
            # 左の要素が右より大きい場合は交換
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    
    return arr

# 使用例
data = [64, 34, 25, 12, 22, 11, 90]
print("ソート前:", data)
bubble_sort(data)
print("ソート後:", data)

改良版（最適化されたバブルソート）

def optimized_bubble_sort(arr):
    """
    最適化されたバブルソート
    交換が発生しなかった場合は早期終了する
    
    引数:
        arr: ソートする配列
    """
    n = len(arr)
    
    for i in range(n):
        swapped = False  # 交換フラグ
        
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True  # 交換が発生
        
        # 交換が発生しなかった場合は完了
        if not swapped:
            break
    
    return arr

# 使用例
data = [1, 2, 3, 4, 5]  # すでにソート済み
print("ソート前:", data)
optimized_bubble_sort(data)
print("ソート後:", data)
print("→ 最適化により早期終了できる")

詳細情報を表示するバブルソート

def bubble_sort_with_info(arr):
    """
    処理の詳細を表示するバブルソート
    
    引数:
        arr: ソートする配列
    """
    n = len(arr)
    comparisons = 0  # 比較回数
    swaps = 0        # 交換回数
    
    print(f"初期配列: {arr}")
    print("-" * 50)
    
    for i in range(n):
        print(f"\n【パス {i + 1}】")
        
        for j in range(0, n - i - 1):
            comparisons += 1
            print(f"  比較: arr[{j}]={arr[j]} と arr[{j+1}]={arr[j+1]}", end="")
            
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swaps += 1
                print(f" → 交換！")
            else:
                print(f" → 交換なし")
        
        print(f"  パス後: {arr}")
    
    print("-" * 50)
    print(f"\n完了！")
    print(f"比較回数: {comparisons}")
    print(f"交換回数: {swaps}")
    print(f"最終結果: {arr}")
    
    return arr

# 使用例
data = [5, 2, 8, 1, 9]
bubble_sort_with_info(data)
